# Romantic Coding Girl

## Overview
A cute animated Valentine's website featuring a pink love truck that drives across the screen and a trembling envelope with a love letter that reveals on hover.

## Project Architecture
- Pure static HTML/CSS website (no JavaScript, no build step)
- `loveTruck.html` / `index.html` — main page
- `loveTruck.css` — truck animation styles
- `envelope.css` — envelope animation styles

## How to Run
Served via Python's built-in HTTP server on port 5000.

## Deployment
Static deployment from the workspace root directory.
